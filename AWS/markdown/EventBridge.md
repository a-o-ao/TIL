# Amazon EventBridge

AWS上のリソースの状態変化やスケジュールに応じてアクションを実行することができるサービス

## API destinations（API送信先）

- 外部APIとの連携を簡単に行うことができる機能
- イベントが発生した際に外部のエンドポイントに対してHTTPSリクエストを非同期に送信することが可能

例えば、S3バケットに新しいオブジェクトが追加されたときに外部の監視サービスに通知を送るなど
