### 配列

## 配列からリストに変換

```
/**
 * 配列からリストへ変換します。
 */
public static void main(String arg[]) {
    String[] array={"その１","その２","その３"};
 
    // ユーティリティークラス 「java.util.Arrays」 を使って、
    // 配列からリストへ変換します。
    List<String> list=Arrays.asList(array);
 
    for (String str : list) {
        System.out.println(str);
    }
}
```
