# 🎯 AWS AI Practitioner 合格に必要な最小知識セット

---

## 🧠 1. 機械学習の超基本（必須）

### 学習方式（最重要）
| 種類 | ひとこと理解 | 代表例 |
|------|------------|-------|
| 教師あり学習 | 正解付きデータで学習 | 分類・回帰 |
| 教師なし学習 | 正解なしで学習 | クラスタリング |
| 強化学習 | 報酬最大化 | ゲームAI |

### 過学習（超重要）
| 用語 | 意味 |
|------|------|
| Overfitting | 学習しすぎ |
| Underfitting | 学習不足 |

## 🤖 2. 生成AI / LLM（最重要分野）

### 基本概念
| 用語 | ひとこと理解 |
|------|------------|
| LLM | 大規模言語モデル |
| Foundation Model | 汎用AIモデル |
| Prompt | AIへの指示 |
| RAG | 検索＋生成 |
| Fine-tuning | 追加学習 |
| Embeddings | ベクトル化 |

### LLMパラメータ（必須3つ）
| パラメータ | 意味 |
|----------|------|
| temperature | 創造性 |
| top-p | 確率範囲 |
| max_tokens | 出力長 |

👉 これだけで8割の生成AI問題に対応可能
